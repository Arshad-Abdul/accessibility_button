📋 Integration Instructions

The fastest way to use this toolbar is the one-line embed below. Manual HTML, CSS, and JavaScript integration is still available further down the page if you want full control.

  1. Paste the one-line embed script into your website
  2. Load the page and the toolbar will appear automatically
  3. Use manual integration below only if you want to customize the full source
💡 Note: This toolbar is fully WCAG 2.1 compliant and uses localStorage to remember user preferences across sessions. All settings persist until the user resets them.

🌟 Features Breakdown

1. Font Size Control

Allows users to increase or decrease text size from 80% to 200% in 10% increments. Perfect for users with low vision.

2. High Contrast Mode

Switches to pure black and white for maximum contrast. Excellent for users with certain vision impairments.

3. Dark Mode

Reduces eye strain in low-light environments by using dark backgrounds with light text.

4. Low Vision Mode

Uses a black background, yellow text, and larger font size to maximize readability for many low-vision users.

5. Large Cursor

Increases cursor size for easier tracking. Helpful for users with motor control issues.

6. Underline Links

Makes all links underlined and bold for better visibility and identification.

7. Highlight Links

Adds yellow background to all links making them impossible to miss.

8. Dyslexia Font

Applies OpenDyslexic with safe fallbacks to improve letter distinction for many readers with dyslexia.

9. Letter Spacing

Increases space between characters for improved readability.

10. Line Height

Increases vertical spacing between lines of text for easier reading.

11. Reading Mode

Keeps a clear horizontal strip near the pointer and dims the rest of the page to support focused reading.

12. Settings Persistence

All settings are automatically saved to localStorage and persist across page refreshes and browser sessions.
